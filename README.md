#  Digital Hand Cricket Game

A virtual recreation of the classic hand cricket game built in C++, featuring multiple teams, customizable match formats, difficulty levels, and interactive gameplay mechanics.

##  Project Overview

Hand Cricket 2024 brings the nostalgic hand cricket experience to the console through an engaging cricket simulation game. Players can select teams, choose match formats, set difficulty levels, and compete in exciting cricket matches with live score tracking and interactive gameplay.

The project was developed to combine programming concepts with game logic and create a fun user experience using C++.

---

##  Features

###  Team Selection
Choose from different international cricket teams such as:
- Pakistan
- India
- Australia
- and more

###  Multiple Game Modes
Play different match formats including:
- Blitz Match
- T5 Match
- Super Over
- Custom Overs Mode

###  Difficulty Levels
Adjust gameplay using:
- Easy
- Medium
- Hard

modes for different skill levels.

###  Live Match Scoreboard
Track:
- Runs
- Wickets
- Overs
- Match Status

in real time during gameplay.

###  Audio & Effects
Integrated background audio and sound effects to make gameplay more interactive.

---

##  Technologies Used

- C++
- Standard Template Library (STL)
- `<iomanip>` for formatting
- `rand()` for match randomization
- System commands for audio integration

---

##  Gameplay Flow

1. Launch the game  
2. Select a game mode  
3. Choose your team  
4. Set difficulty level  
5. Toss simulation  
6. Play batting and bowling turns  
7. View final match results

##  How to Run

### Clone the Repository

```bash
git clone https://github.com/your-username/HandCricket2024.git
```

### Navigate to the Project Folder

```bash
cd HandCricket2024
```

### Compile the Program

```bash
g++ -o handcricket HandCricket2024.cpp
```

### Run the Game

```bash
./handcricket
```
